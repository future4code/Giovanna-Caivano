Projeto: Sistema Casa da Nutrição

Projeto de sistema simples criado para uma clínica de nutrição hipotética durante os cursos iniciais de Javascript da Alura. O formulário valida as informações fornecidas e adiciona os novos pacientes na tabela.

Tecnologias utilizadas:

  • HTML;
  • CSS;
  • Javascript.
